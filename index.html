Mathematical Induction

Reference Paper

Concept of Mathematical Induction


• Technique mostly used for rigrously proving statements

• Used to check the validity of a formula without having
  to check each and every case by hand

Components of Inductive Proof


• Base case - usually $ n = 1 $

• Induction Hypothesis - Strong/Weak (Assume $P(k)$ holds)

• Inductive Step - Prove hypothesis ($P(k) \implies P(k+1)$)

Strong v/s Weak Induction

The difference between weak induction and strong induction appears in the induction hypothesis


• Weak Induction - $P$ is true only at k

• Strong Induction - $P$ is true $\forall \ \ i \ \in \{1,2,3,...,k\}$

Example


Prove the following statement is true for all integers $n$.

$$P:\sum_{i=1}^n i = \frac{n(n+1)}{2}$$

Base Case

We prove that the statement is true for $n = 1$ \[ P(1):\sum_{i=1}^1 i = \frac{1(1 + 1)}{2} = 1\]

Induction Hypothesis

(Weak) Assume the statement to be true for some $n=k$ \[P(k):\sum_{i=1}^{k}i=\frac{k(k+1)}{2}\]

Inductive Step

We prove the statement for $n=k+1$ $$P(k+1):\sum_{i=1}^{k+1}i=\sum_{i=1}^{k}i+(k+1)=\frac{k(k+1)}{2}+k+1=\frac{(k+1)(k+2)}{2}$$ Thus, $P$ is true for all $n$

Weak Induction Example


Show that

$$P: \sqrt{2 + \sqrt{2 + \sqrt{2 + \ ... \ + \sqrt{2}}}} = 2\cos{\frac{\pi}{2^{n+1}}}$$

Here there are $n$ 2s in the expression on the left

Solution


Base Case

We prove that the statement is true for $n = 1$ \[ P(1):\sqrt{2} = 2\cos{\frac{\pi}{2^{2}}} = \sqrt{2}\]

Induction Hypothesis

(Weak) Assume the statement to be true for some $n=k$

\[ P(k): \sqrt{2 + \sqrt{2 + \sqrt{2 + \ ... \ + \sqrt{2}}}} = 2\cos{\frac{\pi}{2^{k+1}}} \]

Inductive Step

Adding 2 and taking square root on both sides

\[ P(k+1): \sqrt{2 + 2\cos{\frac{\pi}{2^{k+1}}}} = 2\cos{\frac{\pi}{2^{k+2}}} \]

Strong Induction Example


Show that every integer greater than or equal to 2 can be factored into primes

Solution


Base Case

We prove that the statement is true for $n = 2$

Since 2 is a prime number, prime factorization of 2 is 2 itself.

Induction Hypothesis

(Strong) Assume for all integers less than or equal to k, the statement holds

Inductive Step

Case 1: k+1 is a prime number

Same as base case, it is prime factorization for itself.

Case 2: k+1 is not a prime number

\[ k + 1 = p \times q \ (p,q \in \mathbb{Z^{+}}) \implies p,q \le k + 1 \]

From the induction hypothesis, both p and q can be factorized into primes.

Defining a Bijection


• A map f between sets $S_{1}$ and $S_{2}$ that is injective and
  surjective.

• $S_{1} \sim S_{2} \implies \exists \ \ f : S_{1} \longrightarrow S_{2}$, where f is a bijection

• For a bijection, $|S_{1}| = |S_{2}|$

Countable Set


• Set of all natural numbers is denoted by $\mathbb{N} = \{1,2,...\}$

• A set S is called countable if $S \sim T$ for some $T \subseteq \mathbb{N}$

   • If $T \subset \mathbb{N}$, S is countably finite

   • If $T = \mathbb{N}$, S is countably infinite
